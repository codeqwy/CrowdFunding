# CrowdFunding

#### Description
尚筹网：一个在线众筹平台通过向普通大众募集资金来支持创业项目，支持者可以获得与支持金额相当的回报。
项目分后台管理系统和前台会员系统两部分：
1. 后台管理系统是基于SSM整合的单一架构
2. 前台会员系统是基于SpringBoot+SpringCloud的微服务架构
